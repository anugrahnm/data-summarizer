# Data Summarizer

An AI-powered document summarisation tool. Upload a PDF or plain text file and get a clean, structured summary generated by Google Gemini. Built with a decoupled architecture: React frontend, FastAPI backend, containerised with Docker, and self-hosted on a DigitalOcean VPS behind a Caddy reverse proxy with automatic HTTPS.

## Live Demo

**Frontend:** [summarizer.anugrah.dev](https://summarizer.anugrah.dev)  
**API:** [summarizer-api.anugrah.dev/docs](https://summarizer-api.anugrah.dev/docs)

## How it works

1. User uploads a PDF or TXT file via the React frontend
2. The frontend sends the file to the FastAPI backend
3. The backend extracts the text and sends it to the Gemini API
4. Gemini returns a structured summary
5. The frontend renders it as clean Markdown

## Project Structure

```
data-summarizer/
backend/ ← FastAPI API (Python, uv)
frontend/ ← React + Vite + TypeScript
docker-compose.yml
```

## Setup

### .env

```
GEMINI_API_KEY=your_key_here
```

### Backend

```bash
cd backend
uv sync
uv run uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Switching environments

For local development, update the API URL in `frontend/src/App.tsx` to `http://127.0.0.1:8000/summarize/` and update the CORS origin in `backend/main.py` to `http://localhost:5173`. For production, use your deployed API and frontend URLs instead.

### Docker

```bash
docker compose up -d --build
```
