# Data Summarizer

An AI-powered document summarisation tool. Upload a PDF or plain text file and get a clean, structured summary generated by Google Gemini.

## Live Demo

**Frontend:** [data-summarizer.streamlit.app](https://data-summarizer.streamlit.app)  
**API:** [Render.com](https://data-summarizer-11ko.onrender.com/docs)

## How it works

1. User uploads a PDF or TXT file via the Streamlit frontend
2. The frontend sends the file to the FastAPI backend
3. The backend extracts the text and sends it to the Gemini API
4. Gemini returns a structured summary
5. The frontend renders it as clean Markdown

## Project Structure

```
data-summarizer/
  backend/    ← FastAPI API (Python)
  frontend/   ← Streamlit UI (Python)
```

## Setup

### .env

```
GEMINI_API_KEY=your_key_here
API_URL=http://127.0.0.1:8000 # or whatever url your backend is hosted at.
```

### Backend

```bash
cd backend
uv sync
```

Run:

```bash
uv run uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
pip install -r requirements.txt
```

Run:

```bash
streamlit run app.py
```
